=== Installation des pré-requis ===

$ mkdir /tmp/cubemedia
$ cd /tmp/cubemedia

Récupérer la bibliothèque Python pour l'upload dans Vimeo

$ git clone https://github.com/dkm/python-vimeo.git
$ cd python-vimeo
$ sudo python setup.py install

=== Exemples ===

$ nano /usr/local/bin/vimeo-uploadv2.py
